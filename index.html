Readonly mode

If the original input field has a readonly attribute, then the Tags will not have a way to add or remove any tags.

HTML

<input name='tags4' readonly value='tag1, tag 2, another tag'>

jQuery plugin version

Tagify can also be used as a jQuery plugin

HTML

<input name='tags-jquery' value='try, adding, a tag'> <button class='tags-jquery--removeAllBtn' type='button'>Remove all tags</button>

JAVASCRIPT

var $elm = $('input[name=tags-jquery]')
    .tagify()
    .on('add', function(e, tagName){
        console.log('added', tagName)
    });

$('.tags-jquery--removeAllBtn').on('click', function(){
    $elm.trigger('tagify.removeAllTags');
})
